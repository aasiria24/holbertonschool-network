# Networking Basics #0

A collection of answer files and Bash scripts covering fundamental networking concepts.

## Topics Covered

- OSI Model
- Types of networks (LAN, WAN, Internet)
- MAC and IP addresses
- TCP and UDP protocols
- TCP/UDP ports
- ICMP and ping

## Files

| File | Description |
|------|-------------|
| `0-OSI_model` | Answer file about the OSI model |
| `1-types_of_network` | Answer file about LAN, WAN, and Internet |
| `2-MAC_and_IP_address` | Answer file about MAC and IP addresses |
| `3-UDP_and_TCP` | Answer file about UDP and TCP protocols |
| `4-TCP_and_UDP_ports` | Bash script that displays listening ports |
| `5-is_the_host_on_the_network` | Bash script that pings an IP address 5 times |

## Usage

### 4-TCP_and_UDP_ports
```bash
sudo ./4-TCP_and_UDP_ports
```

### 5-is_the_host_on_the_network
```bash
./5-is_the_host_on_the_network {IP_ADDRESS}
```
